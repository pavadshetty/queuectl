# QueueCTL - Job Queue Management CLI

This project is a command-line tool that manages a simple job queue system.  
It lets you enqueue jobs, view their status, and process them using a worker.

---

## Features

- Add (enqueue) new jobs to a queue  
- Process jobs asynchronously  
- View completed/pending job statuses  
- Uses SQLite for persistent job storage  
- Built with Python and Click (CLI framework)
